Physics Sandbox Game

A dynamic physics simulation sandbox built with Python and Pygame

Features

Interactive Tools

  • Normal Ball: Standard physics behavior
  • Bouncy Ball: Enhanced elasticity
  • Heavy Ball: Increased mass
  • Light Ball: Reduced mass
  • Wall Tool: Create custom walls and structures
  • Eraser Tool: Remove objects from the simulation

Physics Simulation

  • Realistic ball-to-ball collisions
  • Wall collision detection and response
  • Gravity and friction effects
  • Elasticity and momentum conservation
  • Anti-sticking mechanisms for smooth interactions

User Interface

  • Clean, intuitive tool menu
  • Real-time FPS display
  • Smooth wall drawing with point optimization
  • Visual feedback for tool selection
  • Drag-and-drop ball creation

Technical Features

  • Spatial partitioning for performance optimization
  • Point optimization for wall creation
  • Smooth wall drawing with distance-based sampling
  • Configurable physics parameters
  • Automatic package dependency management

How It Works

  1. Launch the game and select your desired tool from the menu
  2. Use the left mouse button to create and interact with objects
  3. Draw walls by clicking and dragging with the wall tool
  4. Remove objects using the eraser tool
  5. Watch as the physics engine calculates realistic interactions between objects

System Requirements

  • Python 3.x
  • Pygame
  • NumPy
  • Mouse and keyboard for interaction

Project Summary

The Physics Sandbox Game is an interactive physics simulation that allows users to experiment with various physics objects in real-time. Built with Python and Pygame, it features realistic physics calculations, intuitive controls, and a clean user interface. Users can create different types of balls, draw custom walls, and observe how objects interact with each other and the environment. The project is optimized for performance and includes features like spatial partitioning and point optimization for smooth gameplay.
